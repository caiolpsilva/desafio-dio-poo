# Desafio POO - DIO

Modelagem de um iPhone utilizando Programação Orientada a Objetos em Java.

## Funcionalidades

Reprodutor Musical
- tocar
- pausar
- selecionarMusica

Aparelho Telefônico
- ligar
- atender
- iniciarCorreioVoz

Navegador Internet
- exibirPagina
- adicionarNovaAba
- atualizarPagina
